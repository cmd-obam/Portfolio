# Han's Portfolio

1.디자인을 선택한 이유
어떤 기준으로 이 디자인을 선택했는지
참고한 자료나 의도(타깃, 분위기 등)

- 사실 디자인적인 부분에 어려움이 많기 때문에 심플한 디자인으로 결정했습니다.
의도한 느낌은 잡지에서 나올듯한 느낌으로 구현하였습니다.



2.기획 방향
전체 화면/구조를 어떻게 설계했는지
우선순위를 둔 요소는 무엇이었는지

- 화면 구조를 포트폴르오 제목란을 제외하고 총 5개의 큰 컨테이너로 설계했습니다.
상단의 3개는 프로필, 사진, 컨택트를 하단 2개는 워크와 스포츠 , 에듀케이션과 스킬을 한세트로 구성하였습니다.

구성 하면서 제일 우선순위로 둔 요소는 프로필 사진입니다. 이 프로필 사진을 기준으로 좌우의 컨텐츠와 하단의 컨텐츠의 위치와 크기를 세세하게 조절할수 있을수 있었습니다.



3.작업 중 어려웠던 점 (1~2가지)
구현하면서 막혔던 부분
고민이 가장 컸던 포인트

디자인 자체가 단순하기때문에 추가요소를 넣어야 하는 부분으로 배너 슬라이드가 가장 어려웠던거 같습니다. 최초 작성시 이미지가 My hobby 글자를 가로질러올라가는 현상으로 여러번 수정했으나 수정되지 않고 이미지가 가로 질러가게 되엇습니다.

4.해결 과정⭐
문제를 어떻게 접근했고, 어떤 방식으로 해결했는지
타협한 부분이 있다면 그 이유

- 질문을 통해 이미지를 감싼 컨테이너의 위치를 다시 확인하였습니다.
그리고 이미지를 추가함으로 인해 배너 슬라이드 되는 속도에 대하여 조절하였습니다



5. 느낀 점 / 소감
이번 실습을 통해 배운 점
다음에 다시 한다면 바꾸고 싶은 부분

- 만약 다시 만든다고 한다면 추가적인 기능을 넣고싶습니다
예를들면 다크모드인데 사실 구현해보고싶었는데 감을 잡기 어려워서 조금더 배우고 공부한뒤에 해보고 싶습니다.
그리고 움직이는 이미지도 조금 간단하게 구현해서 제목부분을 호버한다거나 할수 있는 그런 방법을 구현해보고 싶습니다
