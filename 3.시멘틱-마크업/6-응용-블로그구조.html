1. 시멘틱 HTML

시멘틱 HTML은 이름 자체가 의미를 전달하는 마크업 방식입니다. div와 span 대신 header, nav, main, article 같은 의미 있는 태그를 사용합니다.

2. 왜 중요한가?

시멘틱 마크업의 3가지 주요 이점:

  • 접근성 향상: 스크린 리더가 페이지 구조를 명확히 파악
  • SEO 최적화: 검색 엔진이 콘텐츠를 정확히 이해
  • 유지보수성: 코드 가독성이 높아 팀 협업에 유리

3. 주요 시멘틱 태그

HTML5에서 제공하는 7개 핵심 시멘틱 태그:

  • header - 헤더 영역
  • nav - 네비게이션
  • main - 주요 콘텐츠
  • article - 독립적인 콘텐츠
  • setction- 주제별 그룹
  • aside - 부가정보
  • footer - 바닥글

4. 실무 적용 팁

실제 프로젝트에서는 시멘트 태그와 div를 적절히 혼용합니다. 의미가 명확한 곳은 시멘틱 태그를, 단순 레이아웃 컨테이너는 div를 사용하세요.
