HTML 문서 구조의 중요성

올바른 HTML 문서 구조는 웹 개발의 기초입니다.

검색 앤진 최적화(SEO)

meta 태그는 검색 엔진이 페이지를 이해하는 데 도움을 줍니다.

  • description: 검색 결과에 표시되는 설명
  • keywords: 페이지의 주요 키워드
  • author: 콘텐츠 작성자 정보

소셜 미디어 공유

Opne Graph 태그를 사용하면 SNS에서 공유할 떄 미리보기가 멋지게 표기됩니다.

  • og:title : 공유될 떄 표시되는 제목
  • og:description : 공유될때 표시되는 설명
  • og:type : 콘텐츠 유형

접근성

lang 속성과 적절한 meta 태그는 스크린 리더 사용자의 경험을 개선합니다.
